News Article Classifier

A Machine Learning based News Article Classification system built using Python and Streamlit.

Features:
- World News
- Sports
- Business
- Science & Technology

Tech Used:
- Python
- Scikit-learn
- Streamlit
- NLP (TF-IDF)

How to Run:
pip install -r requirements.txt
streamlit run news.py
